# Group-video-chat
Group video chat app built using HTML, CSS, JavaScript and the Agora SDK For Student Ambassador Program 

# Installation
* 1 - clone repo https://github.com/abubakarharuna10
* 2 - Create an account on agora.io and get APP ID, Temp Token and Channel Name
* 3 - Update APP ID, Temp Token and Channel Name in script.js
